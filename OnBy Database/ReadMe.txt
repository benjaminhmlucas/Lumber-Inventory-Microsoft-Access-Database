This database prices, tracks, and records Dimensional lumber cut and sold at the lumber mill.  
Price is based on dimension and features.  
Sold table is static so prices remain historically correct.
Price is first calculated with width cost categorie.  This determines the base per board foot cost.  
Then the per board foot feature coast is added for features for that species. 
Then this base price is multiplied by total board feet (Thickness(in)xWidth(in)xLength(ft))/12.
Then that total is multiplied buy number of boards in bundle.
